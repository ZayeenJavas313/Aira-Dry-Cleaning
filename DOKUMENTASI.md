# Dokumentasi Penggunaan Aplikasi Aira Laundry

Dokumen ini berisi panduan penggunaan sistem informasi manajemen Aira Laundry untuk setiap peran pengguna. Setiap bagian menjelaskan alur kerja secara berurutan, mulai dari mengakses sistem hingga keluar.

---

## 1. Admin

| Kategori | Deskripsi |
|----------|-----------|
| **Mengakses Sistem** | Buka browser, akses halaman sistem, login menggunakan akun Administrator. |
| **Kelola User** | Tambah, ubah, dan hapus data pengguna (admin, kasir, pegawai, pemilik). |
| **Kelola Pelanggan** | Tambah, ubah, hapus, dan lihat data pelanggan. |
| **Kelola Transaksi** | Tambah, ubah, dan hapus transaksi laundry. Sistem otomatis menghitung harga sesuai layanan yang dipilih dan menghasilkan ID nota 6 digit. |
| **Kelola Layanan** | Tambah data layanan dan tarif, ubah tarif, serta hapus layanan yang tidak tersedia. |
| **Laporan** | Lihat laporan pendapatan harian dan rekap transaksi dalam periode tertentu. |
| **Logout** | Keluar dari sistem setelah selesai menggunakan aplikasi. |

---

## 2. Kasir

| Kategori | Deskripsi |
|----------|-----------|
| **Mengakses Sistem** | Login menggunakan akun Kasir. |
| **Kelola Pelanggan** | Tambah data pelanggan baru beserta nomor HP, ubah data jika diperlukan, dan lihat data pelanggan yang sudah terdaftar. |
| **Kelola Transaksi** | Catat transaksi laundry baru, ubah jika ada kesalahan input. Sistem otomatis menghitung harga berdasarkan layanan dan berat atau jumlah item, serta menghasilkan ID nota otomatis. |
| **Kelola Pembayaran** | Input dan perbarui status pembayaran (lunas / belum lunas). |
| **Cetak Nota** | Cetak nota transaksi sebagai bukti penerimaan laundry untuk diserahkan kepada pelanggan. |
| **Logout** | Keluar dari sistem setelah selesai digunakan. |

---

## 3. Pegawai Laundry

| Kategori | Deskripsi |
|----------|-----------|
| **Mengakses Sistem** | Login menggunakan akun Pegawai Laundry. |
| **Lihat Daftar Laundry** | Melihat daftar cucian yang sedang diproses beserta detail transaksi yang harus dikerjakan. |
| **Update Status Laundry** | Mengubah status dari **Cucian Masuk → Proses Cuci → Selesai / Harum**. |
| **Logout** | Keluar dari sistem setelah selesai digunakan. |

---

## 4. Pelanggan

| Kategori | Deskripsi |
|----------|-----------|
| **Cek Status Laundry** | Melihat status pengerjaan laundry berdasarkan nomor nota transaksi (masih diproses / selesai / siap diambil). |
| **Konfirmasi Pengambilan Laundry** | Menunjukkan nota atau ID transaksi kepada petugas kasir, melakukan konfirmasi penerimaan, dan menyelesaikan proses pengambilan laundry. |

---

## 5. Pemilik Usaha

| Kategori | Deskripsi |
|----------|-----------|
| **Mengakses Sistem** | Login menggunakan akun Pemilik Usaha. |
| **Dashboard** | Melihat ringkasan bisnis meliputi total pendapatan, total transaksi, jumlah pelanggan, dan jumlah pegawai. |
| **Laporan & Analitik** | Melihat laporan pendapatan (harian, mingguan, bulanan, tahunan) serta grafik transaksi dan layanan terpopuler. |
| **Kelola Pelanggan** | Melihat data pelanggan yang terdaftar. |
| **Kelola Layanan** | Melihat daftar layanan dan tarif yang tersedia. |
| **Kelola Pengguna** | Melihat data pegawai dan kinerja masing-masing. |
| **Logout** | Keluar dari sistem setelah selesai digunakan. |

---

*Dokumen ini disusun untuk memudahkan pengguna dalam mengoperasikan sistem Aira Laundry sesuai dengan peran dan wewenang masing-masing.*
